克隆代码库 clone 
                # 拉取最新代码
                git clone 地址
提交代码 commit

提交前应先拉取服务器上的最新代码，拉取后检查差异代码，
保留正确的代码，删掉错误的代码
养成每天提交代码的习惯

 
                # 拉取最新代码
                git fetch   //不自动合并(merge)差异代码(diff)
                //或
                git pull    //自动合并(merge)差异代码(diff)
                # 把新文件(new)和被修改(modified)文件和删除(deleted)文件添加到暂存区
                git add -A
                # 把暂存区的所有内容提交到当前分支
                git commit -m "some message"
                # 把当前分支的代码推送到服务器
                git push
                # 查看当前所在分支
                git status
                # 查询提交记录
                git log
代码回滚 reset

回滚之后提交过的代码都会被退回到本地暂存区，
可以再次提交，也可以放弃更改

 
                # 回滚到上一版本
                git reset --hard HEAD^   //HEAD就是指当前版本，一个^就是前一个版本
                # 回滚到之前第n个版本
                git reset --hard HEAD~n
                # 回滚到commitid版本
                git reset --hard commitid

                # 为你本地库设置name和email

                git config --global user.name "你的username"  //当前目录git库的name
